Binary Search Tree Notes:

Operations Supported  in a BST :
  • Search,
  • Min,
  • Max,
  • Successor ( node)
  • Predecessor (node)
  • Add ( Normal while creating Tree)
  • Delete
 All the operations are performe in O(h) time where h is the height of tree.
Each operation has a recursive and non recursive version.
